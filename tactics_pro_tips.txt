## Gear replacement

Before dismantling:
- Check if gear can transfer to alts
- Keep niche-use gear for specific content (e.g. resist set)
- Check soul imprint and trade restrictions

Never dismantle immediately if secondary value exists.

---

## Refinement

Meta shifts change refinement priorities. When planning:
- Compare legacy lines vs new set trajectories
- Include kinah burn rate
- Avoid chasing trendy paths without a cost check

A stable moderate path usually beats unstable high-cost spikes.

---

## Rift timing

Rift windows are time-limited. Late arrival = lost rewards. Pre-position before the window opens. Lock rift windows in your weekly planner first, then fill other content around them.

---

## Soul crystals

Soul crystals often pile up unused. Build a routine: pet progression interface, periodic spend. Delayed usage = delayed stat value. Do a soul crystal audit every few sessions.

---

## Mobile key setup

For mobile players:
- Separate survival, movement, main burst, and utility into zones
- Put forced manual skills in reachable thumb arcs
- Avoid frequent layout changes—muscle memory matters

**Device care:** Use lower-intensity loops during high heat. Manage battery and thermal load.

---

## Routine integration

Link tips to your routine:
- Gear review day
- Refinement budget day
- Rift-focused day
- Soul crystal consumption checkpoint
- Mobile layout review checkpoint

Structure prevents drift.

---

## Common mistakes

- **Immediate disposal of replaced gear** → Check alt/niche value first.
- **Refinement without budget guard** → Set a weekly kinah cap.
- **Rift reaction play** → Pre-position before open windows.
- **Hoarding soul crystals** → Add recurring consumption checkpoint.
- **Constant key layout changes** → Keep a stable layout; refine only pain points.
